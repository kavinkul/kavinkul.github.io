On the Subject of Hereditary Base Notation

I'm not really sure why we invented fancy number systems. We now have to deal with our creation, in an explosive way.

See Appendix H for the description of Hereditary Base N Notation.

  • On this module, there are two number displays and 18 buttons with label 0 to 9, A to F, Clear, and Submit.
  • The top display shows non-hereditary base-N number while the bottom display shows the base-N of this number.
  • To solve this module, convert the number from the top display into a hereditary base-K number. The base of this number will not be given on the module.
  • Use the bomb and its edgework information and count the number of true statements in the table below. Add 2 to this number to get the value K.

    Once the hereditary base-K number is calculated, take all the digits (A, B, C, D, E, F, G, H, I, and J are considered as a single digit) in the number and do the following:

    • If non-hereditary and hereditary bases are both odd or even, add all digits together to get the final number.
    • Otherwise, multiply all digits together to get the final number. Modulo it with 10,000 if it exceeds 9,999.
    • Ignore any digit 0 as it will not be used in any of the calculation.

    The final number must be submitted in non-hereditary base-L number where L is the difference between the number of false statements and the displayed base with negative sign dropped. If the number L is less than 2, then submit the final number in base 10. If the number L is greater than 16, then submit the final number in base 8.

    Submitting the correct number will solve the module while submitting an incorrect number will incur a strike.

    A hereditary base N notation imposes an additional requirement that every number, except the base of an exponent, must be less than the base itself. This means that each of the exponent must be converted into base N number. If, after conversion, the exponent of an exponent is still greater than or equal to the base N, this exponent must also be converted to a base-N number. Repeat this process until every digit that is not the base is less than the base itself.
